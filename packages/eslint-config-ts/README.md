# `@guanghechen/eslint-config-ts`


# Install

* npm

  ```npm
  npm install --save-dev @guanghechen/eslint-config-ts
  ```

* yarn

  ```npm
  yarn add --dev @guanghechen/eslint-config-ts
  ```

# Usage

eslint config for ts + jest project.

  * Use in .eslintrc.js

    ```javascript
    {
      extends: ['@guanghechen/ts'],
    }
    ```
